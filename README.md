# Sales-Analysis
Sales Performance Analysis using Python and Tableau. Analyzed Superstore dataset to identify sales trends, profit margins, and top-performing products. Includes visualizations and an interactive dashboard for data-driven business insights.

Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Tableau (dashboards)
- Google Colab

Steps Performed
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Sales Trend, region and Product Performance Visualization  
4. Created an Tableau Dashboard
